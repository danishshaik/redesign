Why Budicon doesn’t include the 24/48px for size variation?

Since Budicon adapt line style, changing 16px into 24px or 32px into 48px need a huge style adjustment and it will cost unconsistent style with the rest set. Therefore, I consider to release 24/48px in other package set in the future.

"Can I use Budicon in templates and themes?"

Yes, but don’t redistribute the whole set in your templates or themes. And also don’t include the vector files.

How to convert SVG into Icon Font?

All you need is SVG file that included in Asset and convert it using 3rd party Icon Font Generator. (Budicon is temporary using 3rd party Icon Font Generator)

Recommended Icon Font Generator:

Fontastic - http://fontastic.me/

Icomoon - http://icomoon.io/

Follow these step to generate your selected icon using Fontastic:

1. Go to Fontastic website and signup/login.

2. Select import icon and select the "budicon.svg" file in SVG folder.

3. After 850 icons uploaded, select icons you want to use in your project.

4. Select customize, here you can bind character mapping and css class maping.

5. Once you done, download the files (you'll get the proper files for your website project).
